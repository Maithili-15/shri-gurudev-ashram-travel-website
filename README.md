# Shri Gurudev Ashram — Spiritual Travel Platform

Monorepo structure:

| Folder | Purpose |
|--------|---------|
| `Frontend/` | React + Vite admin dashboard (primary UI) |
| `Backend/` | Express API placeholder (not connected yet) |
| `stitch_extracted/` | Design reference from Stitch export |

## Run the admin panel

```bash
cd Frontend
npm install
npm run dev
```

Login with any email and a password of 6+ characters (demo mode).

See [Frontend/README.md](./Frontend/README.md) for full documentation.
